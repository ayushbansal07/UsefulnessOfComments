Welcome to Comments Survey

About this Task -

Programmers sometimes write comments which do not help  to understand the code at all!

We want your opinion for some the comments

For each comment, tell us -

  • Is the comment relevant, useful, helps to read  / understand the code? (USEFUL)
  • Is the comment redundant and not required ? (USELESS)
  • Somewhere in between (PARTIALLY USEFUL)
  • Why do you think so ? A one liner please
  • If you would have written a comment for that portion of the code, what would you have written? Please let us know

Some Instructions before you start -

  • You will be provided a whole code. There will be a button next to some comments for which we require your opinion. Click on the button and a small form asking the above questions will open. Submit your response for that comment in the box. Repeat it for all comments.
  • In addition to the comments review fields, the box also contains field asking your details (name, etc.) You just need to fill these details for the first box you submit. For all subsequent boxes these details would be already filled up for you :)
  • For multi-line comments, the button would be visible only at the first line. You need to provide us with your review for the entire comment. It is guaranteed that the multi-line comment would be within /* */ (for c, cpp) or “”” … “”” for (py).
  • The last comment which is required for you to review would be marked as (LAST). You can stop scrolling downwards from there onwards :)
